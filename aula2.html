Aula 2: Declaração de Variáveis e Comandos de Entrada e Saída

Nessa aula você pode ter acesso a várias explicações que irão te auxiliar a aprender mais sobre declaração de variáveis e comandos de entrada e saída em linguagem C.

Como Funcionam as Declarações de Variáveis?

Ao criar uma variável, dizemos que ela foi declarada. A declaração de uma variável é essencial na primeira vez que a variável é usada. Para fazer essa declaração, basta escrever o tipo da variável, seguido do nome desejado para ela, e finalizar a instrução com um ponto-e-vírgula (;).

Comandos de Entrada e Saída

A entrada de dados possibilita que um programa receba informações de dispositivos de entrada. Para executar o comando de entrada, é essencial incluir a diretiva #include <stdio.h>. O comando scanf() utiliza caracteres de formatação para indicar os tipos de dados dos valores que devem ser inseridos.

Saída de Dados

Podemos imprimir, além de texto puro, o conteúdo de uma variável utilizando a função printf() da biblioteca stdio.h.

Principais Especificadores de Formato

  • %i - inteiro
  • %f - números quebrados
  • %c - caracteres
Declaração de Variáveis

Uma variável é um espaço na memória destinado ao armazenamento de um tipo específico de dado. Toda variável precisa ter um tipo e um nome. O tipo de dado de uma variável define o que ela pode armazenar.

OS TIPOS DE DADOS MAIS USADOS SÃO:
  • int: Usado para números inteiros; Exemplo: 1, 2, -10, 412;
  • float: Usado para números reais; Exemplo: 3.5, -9.7, 24.3;
  • char: Usado para caracteres; Exemplo: a, b, c;
